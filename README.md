# 🎓 Edumanager - Versão Corrigida

Sistema acadêmico simples para gestão de alunos, notas e relatórios.

---

## 🔑 Acesso ao sistema

Login: admin  
Senha: 1234

---

## 📌 Funcionalidades

- Login de acesso ao sistema
- Cadastro de alunos com nome, matrícula, turma e notas
- Cálculo de média
- Definição correta de status (Aprovado/Reprovado)
- Remoção de alunos com atualização completa do sistema
- Dashboard com indicadores gerais
- Relatórios com estatísticas atualizadas
- Validação de entrada de dados

---

## ✅ Comportamento esperado (testes validados)

### 📊 1. Aprovação do aluno
- A média do aluno é corretamente calculada
- O status é definido com base na média final

---

### 🔗 2. Remoção de aluno
- Ao remover um aluno, todas as áreas do sistema são atualizadas corretamente

---

### 👤 3. Validação de notas
- O sistema não permite notas menores que 0 ou maiores que 10
- Mensagem de erro é exibida quando necessário

---

Abrir o arquivo `index.html` em um navegador ou acessar a versão hospedada no Vercel.
